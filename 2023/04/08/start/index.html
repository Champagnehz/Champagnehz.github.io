The New Start

​ 时隔一年半，已经面临毕业，想简短的总结下自己的这段经历。

​ 首先是对自己大学生活的总结：

​ （1）强调一个志同道合的哥们Gayhub，即将成为顶级大牛的摆烂胖球。

​ （2）加速的时间：疫情绑走了我的两年大学时光，也带走了我的编程技术HAHA

​ （3）追逐梦想——考研&&杂学

​ 到了现在，即将快毕业的我认识到什么

    第八届“东方财富杯”全国大学生金融挑战赛c类国赛国家一等奖
    荣获“三好生”
    荣获一级“劳动之星”
    获得“通讯&安全”个人优秀奖
    荣获校级奖学金
    网络安全大赛一等奖
    网络安全大赛三等奖

都只是一个没有谈资的经历，而可能对我有帮助的是通过了CET4、CET6。

其次是未来的迷茫，这是我决定学些什么，记录成长blog的重要因素之一。

​ 自去年2022年底以来，ChatGPT已经掀起了AI革命，这是否是新的风口还是断崖绝唱？目前的情况是比较清晰的。和好兄弟的闲谈也让我意识到，或许停滞的人们真有可能被取代？他期待的赛博女儿我相信是有可能成为现实的。

​ 总之开始更新自己的blog，以此督促自己的学习！

​ 山高水长路漫漫，行路人常应有信心。
